ProQyz — Explanation Upload

Paste explanation JSON for existing questions. This edits questions via the Explanation tab — it does NOT create quizzes, passages, or questions. Run node bin/upload.js <file> --explanation or use the Upload button below.

Full example — Cam 17 Reading Test 01, Passage 1, Questions 36-37 
{
  "mode": "explanation",
  "testTitle": "Cam 17 Reading Test 01",
  "existingQuizUrl": null,
  "passages": [
    {
      "passage": 1,
      "questionGroups": [
        {
          "range": "36-37",
          "questionNumberStart": 36,
          "questionNumberEnd": 37,
          "explanations": [
            {
              "questionNumber": 36,
              "content": "<p>The answer is <b>population</b> because the passage states that the city's population grew rapidly.</p>"
            },
            {
              "questionNumber": 37,
              "content": "<p>The answer is <b>suburbs</b> because the text mentions expansion into surrounding areas.</p>"
            }
          ]
        }
      ]
    }
  ]
}
Field reference 
mode            "explanation" (required)
testTitle       Quiz title as shown in ProQyz (required)
existingQuizUrl Direct URL to quiz edit page (optional)
passages[]      Array of passages to process (required)
  passage       1-based passage number (required)
  questionGroups[]  Array of question groups (required)
    range               "36-37" label (required)
    questionNumberStart Start of range (required)
    questionNumberEnd   End of range (required)
    explanations[]      Array of explanation slots (required)
      questionNumber  IELTS question number (required)
      content         Explanation HTML/text (required)
      slot            Auto-calculated if omitted
                      slot = questionNumber - questionNumberStart + 1
Slot mapping: For "Questions 36-37", Q36 maps to slot 1 and Q37 maps to slot 2. Slot numbers are auto-calculated from questionNumber - questionNumberStart + 1.
