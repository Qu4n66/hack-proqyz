Reference: block shapes

Fill-up (grouped; one block per IELTS "Questions 1-6") 
{
  "type": "fill-up",
  "title": "Questions 1-6",
  "instructions": "Complete the notes below. Choose ONE WORD ONLY from the passage.",
  "content": "The {answer} of London increased between 1800 and 1850. The tunnel was covered with {answer}.",
  "answers": ["population", "soil"]
}
Radio (grouped MCQ; one block per IELTS "Questions 36-40") 
{
  "type": "radio",
  "title": "Questions 36-37",
  "instructions": "Choose the correct letter, A, B, C or D.",
  "questions": [
    {
      "number": 36,
      "text": "What is the reviewer's main purpose?",
      "options": [
        { "label": "A", "text": "to describe what happened during the Battle of Worcester" },
        { "label": "B", "text": "to give an account of the circumstances leading to Charles II's escape" }
      ],
      "answer": "B"
    },
    {
      "number": 37,
      "text": "Why does the reviewer include examples?",
      "options": [
        { "label": "A", "text": "to explain how close Charles II came to losing his life" },
        { "label": "B", "text": "to suggest that Charles II's supporters were badly prepared" }
      ],
      "answer": "A"
    }
  ]
}
Single-question radio (no range) 
{
  "type": "radio",
  "title": "Question 7",
  "instructions": "Choose the correct answer.",
  "questions": [
    {
      "number": 7,
      "text": "What is the reviewer's main purpose?",
      "options": [
        { "label": "A", "text": "..." },
        { "label": "B", "text": "..." }
      ],
      "answer": "B"
    }
  ]
}
How normalization works:
  • type → proqyzType (fill-up→fill_up, radio→radio)
  • instructions → instruction
  • title → displayTitle; number range derived from questions[].number or answers.length
  • questionTypeIndex derived from proqyzType
  • fill-up: only the count of {answer} placeholders is checked (must equal answers.length); the inner placeholder text is not required to match answers[i]
